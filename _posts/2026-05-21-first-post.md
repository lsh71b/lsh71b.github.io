# GitHub Pages 블로그 시작하기

오늘은 GitHub Pages와 Chirpy 테마를 이용해 개인 블로그를 만들기 시작함.

## 만든 이유

운영체제, C언어, 암호학, 보안법 공부 내용을 정리하기 위해 블로그를 만들기로 함.

## 앞으로 올릴 글

- 운영체제 실습 정리
- C언어 프로젝트 기록
- 암호학 개념 정리
- 개인정보보호법과 기술법 관련 글
